### 问题参考

1.关于dns泄露: 我更倾向于减少国外流量获取到你的dns，是的这个问题理论是上无法完美解决的(或许你可以自建dns查询)

2.tun模式下无法劫持局域网dns：推荐手动替换设备对于wifi/宽带的dns，如0.0.0.0、223.5.5.5 (::、2400:3200::1)

3.为什么推荐fakeip模式：多代理节点时dns查询缓存与节点不相近

4.关于测速链接方面：延迟仅供参考，仅代表访问测速链接的延迟
