### 科普

**专线**: IPEL/IPLC!!! 昂贵质量好，抖动小(1ms~10ms)，理论IPEL>IPLC

**中转**: 多线转发!!! 略贵，高峰期可用性高，缺点人多掉速

**直连**: 单线直连!!! 廉价线路，高峰期必炸，稳定性欠佳

**CDN**: 机场为了抗攻击/加速破线路常用cloudflare cdn。本质上你是在用cf连接世界，可通过IP查询如[ip.sb](https://ip.sb)判断节点是否使用cf。

**CF线路**: 套用免费版及“自选ip”算是“伪中转”，该卡还卡；付费Argo路由(按量付费,挺贵)，质量较好，晚高峰可用，一般只出现在略贵的机场；至于昂贵的Enterprise套餐，普通机场根本不可能有
